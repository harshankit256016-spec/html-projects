Simple 3-Page Bootstrap Website

This is a basic web project built using Bootstrap that includes three main pages:

Home Page

Registration Page

Login Page

It is designed as a beginner-friendly project to demonstrate simple front-end development using HTML, CSS, and Bootstrap.

📁 Project Structure
project-folder/
│── index.html        # Home Page
│── register.html     # Registration Page
│── login.html        # Login Page
│── css/
│     └── styles.css  # Custom styles (optional)
│── js/
│     └── script.js   # Custom scripts (optional)
│── assets/           # Images or other assets
🚀 Features

Responsive design using Bootstrap

Clean and simple UI

Navigation between pages

User-friendly registration form

Basic login form layout

🛠️ Technologies Used

HTML5

CSS3

Bootstrap 5

📄 Pages Description
🏠 Home Page (index.html)

Landing page of the website

Contains navigation bar

Links to Login and Registration pages

📝 Registration Page (register.html)

Form for new users to sign up

Includes fields like:

Name

Email

Password

Confirm Password

🔐 Login Page (login.html)

Form for existing users to log in

Includes:

Email

Password
